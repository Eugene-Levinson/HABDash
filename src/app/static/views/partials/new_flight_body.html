Whats a payload doc?

A payload doc (or a flight doc) is a json file that tells the HABDash server how to process your flight data. It allowes you to configure which data you would like to be charted and later on other configuration will also be avalibale throught the payload doc

Do I need a payload doc?

Yes, without a payload doc no flight page will be generated.

Example of a payload doc 

 
            
{
    "flight_name": "tumble",
    "description": "Just another test flight",
    "lat_field_name": "Lat",
    "lon_field_name": "Lon",
    "fields": [
        {
            "field_name": "Flight name",
            "field_type": "str",
            "chart_data": false,
            "field_order": 0
        },

        {
            "field_name": "Lat",
            "field_type": "str",
            "chart_data": true,
            "field_order": 1
        },

        {
            "field_name": "Lon",
            "field_type": "str",
            "chart_data": true,
            "field_order": 2
        },

        {
            "field_name": "Alt",
            "field_type": "int",
            "chart_data": true,
            "field_order": 3
        },

        {
            "field_name": "Temp",
            "field_type": "int",
            "chart_data": false,
            "field_order": 4
        }
    ]
}
