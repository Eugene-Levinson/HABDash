Editing a flight

You can edit your flight configuration by submitting a new payload doc. This will overwrite the old one. Submitting new doc will not cause you to lose your old data. As of right now A any new data will be parsed according to the new doc however any old data will not be re-parsed.

You can edit any aspect of your flight configuration appart from the flight name. This feature will be added later.

For more info please read the documentation.
